# **App Name**: Laugh Factory

## Core Features:

- AI Joke Generation: Generate jokes based on the selected category (dad jokes, dark humor, pick-up lines) using a pre-trained model. The LLM will use its tool to filter the generated jokes, so only safe-for-work humor is shown, unless the user explicitly sets a preference.
- Joke Display: Display the generated joke in a clean, user-friendly interface.
- Category Selection: Allow users to select from available joke categories.
- Joke Sharing: Implement a simple UI to allow users to copy the generated joke to the clipboard for easy sharing.

## Style Guidelines:

- Primary color: A cheerful and inviting coral (#FF7F50) to evoke feelings of warmth and amusement.
- Background color: Light beige (#F5F5DC), providing a soft and unobtrusive backdrop to ensure readability and focus on the jokes.
- Accent color: A lively sky blue (#87CEEB), which is analogous to the coral but stands out to highlight interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern look.
- Use flat, minimalistic icons related to joke categories and sharing options.
- Maintain a clean and straightforward layout with clear visual hierarchy to improve user experience.
- Implement subtle animations on joke generation and category selection for enhanced interactivity.
